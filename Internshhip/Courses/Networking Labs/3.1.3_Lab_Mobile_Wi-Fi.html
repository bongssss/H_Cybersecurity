Complete Lab - Mobile Wi-Fi

Introduction

In this lab, you will turn the Wi-Fi radio on and off, forget a found Wi-Fi network, and find and connect to a Wi-Fi network.

Recommended Equipment

  • Android tablet or smartphone running Android version 5.0 or higher
  • iOS tablet or smartphone running iOS version 7.0 or higher

Instructions

Step 1: Configure Wi-Fi on Android device

In this step, you will configure Wi-Fi on an Android device.
  1. Turn on the device and log in with the password, pin code, or another passcode, if necessary.
  2. Go to All apps > Settings > Connections.
  3. Touch the Wi-Fi slider to turn it Off.
  4. Touch the Wi-Fi slider again to turn it On.
  5. Wait until the device is connected to a wireless network before moving on to the next step.
  6. Touch Wi-Fi and then the name of the network to which the device is connected. The Wi-Fi details window appears.
  7. Touch FORGET.
Question:
What happens?
Answer Area
The network to which the device was connected is in the list of available networks, but the device is no longer connected to it.
  1. Touch the network to which the device used to be connected.
  2. Type the Wi-Fi password. Touch CONNECT.
Question:
What happens?
Answer Area
The device connects to the Wi-Fi network.

Step 2: Configure Wi-Fi on an iOS Device.

In this step, you will configure Wi-Fi on an iOS device.
  1. Turn on the device and log in with the password, PIN code, or another passcode, if necessary.
  2. Go to Settings > Wi-Fi.
  3. Touch the Wi-Fi slider to turn it Off.
  4. Touch the Wi-Fi slider again to turn it On.
  5. Touch the name of the network to which the device should connect to. The Wi-Fi details window opens.
  6. Touch Forget This Network. Confirm the deletion of the network by selecting Forget.
Question:
What happens?
Answer Area
The network to which the device was connected is in the list of available networks, but the device is not connected to it.
  1. Touch the network to which the device used to be connected to.
  2. Type the Wi-Fi password.
  3. Touch Join.
Question:
What happens?
Answer Area
The device connects to the Wi-Fi network.
